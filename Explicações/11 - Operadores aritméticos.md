# Operadores aritméticos profº Thiago Leite

São símbolos especiais quais são capazes de realizar ações especificas em um, dois ou mais operadores, em seguida, retornar um resultado.

### Tipos:

- pós-fixado: exp++ ou exp -- // soma uma unidade e subtrai uma unidade, de uma forma tardia. Faz a expressão e depois somar ou subtrair.
- prefixado: ++exp ou --exp //  primeiro ele soma ou subtrai para depois avaliar a expressão.
- aritmético: +, -, *, / e %
- atribuição: =, +=, -=, *=, /= e %=



Ex:

int i = ++k; -> i = k+1

int j = k--; -> j = k; k = k-1;

float = 1.5f + 4.5f;

long l = 1000L * 5L;

double d = f;

i += 5; -> i = i + 5;

j -= 3; -> j = j -3; // sever para o restante *, /, %
